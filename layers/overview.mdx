ShadowBroker aggregates data from dozens of public open-source feeds and renders them as independently toggleable map layers. Every layer can be turned on or off from the left panel without reloading the page.

## API key requirements

Most layers require no API key. The only key required for full functionality is the aisstream.io key for maritime tracking.

| Key | Purpose | Required |
|---|---|---|
| `AIS_API_KEY` | Maritime vessel tracking via aisstream.io WebSocket | Yes |
| `OPENSKY_CLIENT_ID` / `OPENSKY_CLIENT_SECRET` | Higher rate limits for flight data | Optional |
| `LTA_ACCOUNT_KEY` | Singapore LTA CCTV cameras | Optional |

<Note>
  Layers that use no API key — satellites, earthquakes, NASA FIRMS, KiwiSDR, GDELT, DeepState Map, and most others — are fully functional without any configuration.
</Note>

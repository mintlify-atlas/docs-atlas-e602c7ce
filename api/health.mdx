`GET /api/health`

Returns a snapshot of system health including live record counts for each data source, per-source freshness timestamps, and total server uptime. Useful for monitoring dashboards and liveness probes.

<Info>
Rate limit: **30 requests per minute** per IP.
</Info>

## Response

<ResponseField name="status" type="string" required>
  Always `"ok"` while the server is running.
</ResponseField>

<ResponseField name="last_updated" type="string">
  ISO 8601 UTC timestamp of the last completed fast-tier data update.
</ResponseField>

<ResponseField name="sources" type="object" required>
  Current record counts for each data source.
  <Expandable title="properties">
    <ResponseField name="flights" type="number">Number of commercial flights currently tracked.</ResponseField>
    <ResponseField name="military" type="number">Number of military aircraft.</ResponseField>
    <ResponseField name="ships" type="number">Number of AIS vessel positions.</ResponseField>
    <ResponseField name="satellites" type="number">Number of satellite positions computed.</ResponseField>
    <ResponseField name="earthquakes" type="number">Number of recent earthquake events.</ResponseField>
    <ResponseField name="cctv" type="number">Number of live CCTV camera positions.</ResponseField>
    <ResponseField name="news" type="number">Number of news items in the current cluster.</ResponseField>
    <ResponseField name="uavs" type="number">Number of detected UAV positions.</ResponseField>
    <ResponseField name="firms_fires" type="number">Number of active fire detections from FIRMS.</ResponseField>
    <ResponseField name="liveuamap" type="number">Number of LiveUAMap conflict events.</ResponseField>
    <ResponseField name="gdelt" type="number">Number of GDELT geopolitical features.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="freshness" type="object" required>
  ISO 8601 timestamp per data source indicating when it was last successfully fetched. Keys match source names (e.g. `"flights"`, `"ships"`, `"news"`).
</ResponseField>

<ResponseField name="uptime_seconds" type="number" required>
  Total seconds the server process has been running since last start.
</ResponseField>

## Example

```bash
curl http://localhost:8000/api/health
```
